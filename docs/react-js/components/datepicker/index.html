DatePicker

Componente de DatePicker para selecionar uma ou duas datas.

  • Obs.: Todos os exemplos são editávies.
  • Uso
  • Propriedades

Importação​

import { DatePicker } from "@stardust-ds/react";

Uso​

Resultado
Loading...
Editor em tempo real
<DatePicker />

DatePicker hasEndDate.​

Use a propriedade hasEndDate poder selecionar data de início e fim.

Resultado
Loading...
Editor em tempo real
<DatePicker hasEndDate />

DatePicker com valores default​

Resultado
Loading...
Editor em tempo real
<DatePicker
  hasEndDate
  defaultValue={{ start: "2023-01-23", end: "2024-01-23" }}
/>

DatePicker com initialDate​

Use a propriedade initialDate para poder iniciar o picker na data X.

Resultado
Loading...
Editor em tempo real
<DatePicker initialDate="2024-01-10" />

DatePicker com locale​

Use a propriedade locale para alterar o idioma do componente.

Resultado
Loading...
Editor em tempo real
<DatePicker locale="enUS" />

DatePicker com weekDays​

Use a propriedade weekDays para alterar os dias da semana.

Resultado
Loading...
Editor em tempo real
<DatePicker weekDays={["1º", "2º", "3º", "4º", "5º", "6º", "7º"]} />

DatePicker com propriedades inputStartProps e inputEndProps`​

Use as propriedades inputStartProps e inputEndProps para alterar propriedades do DatePicker. Obs.: Aceita praticamente todas as props do componente Input.

Resultado
Loading...
Editor em tempo real
<DatePicker
  inputStartProps={{
    label: "Start Date",
    hasError: true,
  }}
  inputEndProps={{
    label: "End Date",
  }}
  hasEndDate
/>

DatePicker customizado​

Você também pode criar seu próprio DatePicker customizado.

Resultado
Loading...
Editor em tempo real
function example() {
  const style = {
    datePickerContainerStyle: {
      backgroundColor: "#13161A",
      borderRadius: 0,
      borderColor: "#5A646E",
    },
    dayContainerStyle: {
      selected: {
        backgroundColor: "#FFAE00",
        borderColor: "#FFAE00",
        borderWidth: 0,
        borderRadius: 0,
      },
      today: {
        outlineColor: "#FFAE00",
        color: "#FFAE00",
        outlineWidth: 1,
        outlineOffset: -1,
        outlineStyle: "solid",
        borderRadius: 0,
      },
    },
    dayBetweenRangeContainerStyle: {
      backgroundColor: "#FFAE0033",
    },
    dayTextStyle: {
      sameMonth: { color: "#F2F5F8" },
      differentMonth: { color: "#ACB4BA" },
      selected: { color: "#FFFFFF" },
      today: {
        color: "#FFAE00",
      },
    },
    weekTextStyle: {
      color: "#ACB4BA",
    },
    headerTextStyle: {
      color: "white",
    },
    headerIndicatorStyle: {
      color: "white",
    },
    headerActionButtonStyle: {
      arrowContainer: {
        borderRadius: 0,
        borderColor: "white",
      },
      arrowIcon: {
        color: "white",
      },
    },
  };


  return <DatePicker hasEndDate style={style} />;
}

import { DatePickerProps, DatePickerRef } from "@stardust-ds/react";

Propriedades​

type StartEndDate = { start?: string | Date; end?: string | Date };

type OmittedInputProps = Omit<
  InputProps,
  "value" | "defaultValue" | "onChange"
>;

// Obs.:  Para criação de ref. Ex.: useRef(DatePickerRef)
type DatePickerRef = {
  setValues: (dates: Date[]) => void;
  value: Date[];
};
