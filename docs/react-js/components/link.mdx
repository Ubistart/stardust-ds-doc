## Importação

```tsx
import { Link } from "@stardust-ds/react";
```

## Uso

```tsx live
<Link label="Stardust" />
```

### Link externo

```tsx live
<Link
  isExternal
  label="Stardust doc"
  href="https://ubistart.github.io/stardust-ds-doc/"
/>
```

### Link combinado com componente de `Typography`

```tsx live
<Typography type="p1">
  {" "}
  Utilize
  <Link label="link" href="#" /> junto a textos
</Typography>
```

### Utilizando com biblioteca de roteamento

Para usar o Link com uma biblioteca de roteamento como React Router, tudo que você precisa fazer é passar a propriedade `as`.
Ele substituirá a tag renderizada pelo Link de React Router.

```tsx
import { Link as RouterLink } from "react-router-dom";

<Link as={RouterLink} label="Dashboard" to="/dashboard" />;
```

### Link customizado

É possível customizar o label através da propriedade `labelTypographyProps`.

```tsx live
<Link
  isExternal
  label="Stardust doc"
  href="https://ubistart.github.io/stardust-ds-doc/"
  labelTypographyProps={{ color: "red", fontSize: "lg" }}
/>
```
