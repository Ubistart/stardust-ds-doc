Box é um componente que vem com abreviações de estilos bem úteis. Ele renderiza um elemento `div`.

- **Obs.:** Todos os exemplos são editávies.

<Tabs groupId='box-component'>
<TabItem value='Uso' label='Uso'>

## Importação

```tsx
import { Box } from "@stardust-ds/react";
```

## Uso

O componente Box é útil porque ajuda em três casos de uso comuns:

```tsx live
<Box bgColor="orange" color="black" w="100%" px="xxxs" py="xxxs">
  This is the Box
</Box>
```

### Exemplo de card.

```tsx live
function BoxExample() {
  const property = {
    imageUrl: "https://bit.ly/2Z4KKcF",
    imageAlt: "Vista da casa moderna com piscina",
    bedrooms: 3,
    bathrooms: 2,
    title: "Alguma descriação bem legal dessa casa.",
    formattedPrice: "R$2,200.00",
    reviewCount: 34,
    rating: 4,
  };

  return (
    <Box
      maxWidth={380}
      width="100%"
      bWidth="hairline"
      bRadius="sm"
      overflow="hidden"
      bColor="#ddd"
      bgColor="white"
    >
      <img
        width={380}
        height={250}
        src={property.imageUrl}
        alt={property.imageAlt}
      />

      <Box py="xxxs" px="xxxs">
        <Box display="flex" alignItems="baseline">
          <Box color="black" fontWeight="semibold" fontSize="xs" mLeft="quarck">
            {property.bedrooms} Quartos &bull; {property.bathrooms} banheiros
          </Box>
        </Box>

        <Box
          mTop="xxxs"
          color="black"
          fontWeight="semibold"
          as="h4"
          lineHeight="tight"
        >
          {property.title}
        </Box>

        <Box color="black">{property.formattedPrice}</Box>

        <Box mTop="xxxs" color="#4A5568" fontSize="xxxs">
          {property.reviewCount} reviews
        </Box>
      </Box>
    </Box>
  );
}
```

### Propriedade `as`

- Ao inspecionar o elemento a seguir, você econtrará um elemento `button` em vez de um elemento `div`.

```tsx live
<Box
  as="button"
  bRadius="xs"
  bgColor="orange"
  color="white"
  px="xxxs"
  border="none"
  height={36}
  cursor="pointer"
>
  Button
</Box>
```

import { BoxProps } from "@stardust-ds/react";

### Propriedades

- Suporta todas as propriedades de um elemento `div` do HTML.
- Aceita as propriedades de cores (bgColor, color), borda (bWidth, bColor, bRadius), etc...
- Aceita as propriedades de espaçamento e tamanho (width, height, minWidth, maxWidth, minHeight, maxHeight, px, py, mx, my, etc...).
