## Uso

Exemplo de uso básico do componente de checkbox:

```tsx live
function example() {
  const [isChecked, setIsChecked] = useState(true);
  return (
    <div className="row" style={{ alignItems: "center" }}>
      <Checkbox
        checked={isChecked}
        // FIXME: Ajustar onChange do checkbox (devolver true/false)
        onChange={() => setIsChecked(!isChecked)}
      />
      <Typography type="l1">Checkbox</Typography>
    </div>
  );
}
```

### Checkbox customizado

Você também pode criar seu próprio checkbox customizado.

```tsx live
function example() {
  const [isChecked, setIsChecked] = useState(true);
  return (
    <div className="row">
      <div className="row" style={{ alignItems: "center" }}>
        <Checkbox
          checked={isChecked}
          activeColor="red"
          inactiveColor="red"
          iconColor="black"
          onChange={() => setIsChecked(!isChecked)}
        />
        <Typography type="l1">Checkbox</Typography>
      </div>
      <div className="row" style={{ alignItems: "center" }}>
        <Checkbox
          checked={isChecked}
          iconColor="red"
          onChange={() => setIsChecked(!isChecked)}
        />
        <Typography type="l1">Checkbox</Typography>
      </div>
    </div>
  );
}
```
