Skeleton é um componente que vem com abreviações de estilos bem úteis. Ele renderiza um elemento `div`.

- **Obs.:** Todos os exemplos são editávies.

<Tabs groupId='skeleton-component'>
<TabItem value='Uso' label='Uso'>

## Importação

```tsx
import { Skeleton } from "@stardust-ds/react";
```

## Uso

O componente Skeleton é útil para criar um espaço reservado enquanto o conteúdo real está sendo carregado para a página.

### Exemplo de utilização definindo a altura.

```tsx live
<Flex flexDir="column" gap={5}>
  <Skeleton height={20} bRadius="xs" />
  <Skeleton height={20} bRadius="xs" />
  <Skeleton height={20} bRadius="xs" />
</Flex>
```

### Realizando wrap de um componente já existente.

```tsx live
<Skeleton width="fit-content" bRadius="xs">
  <Button>Conteúdo</Button>
</Skeleton>
```

### Alterando as cores

```tsx live
<Skeleton height={20} bgColors={["red", "green"]} bRadius="xs" />
```

### Apresentando o conteúdo quando o conteúdo já foi carregado.

```tsx live
<Skeleton isLoaded bRadius="xs">
  <Button>Conteúdo</Button>
</Skeleton>
```

### Botão de toggle para apresentar o conteúdo

```tsx live
function skeletonExemple() {
  const [isLoaded, setIsLoaded] = useState(false);

  return (
    <Box>
      <Skeleton isLoaded={isLoaded} bRadius="xs">
        <Typography type="h4">Conteúdo legal</Typography>
      </Skeleton>
      <Button mTop="xxxs" onClick={() => setIsLoaded((state) => !state)}>
        Toggle
      </Button>
    </Box>
  );
}
```
