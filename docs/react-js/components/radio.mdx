## Uso

Exemplo de uso básico do componente de radio:

```tsx live
function example() {
  const [value, setValue] = useState("1");

  function handleChange(e) {
    setValue(e.currentTarget.value);
  }

  return (
    <div className="column">
      <div className="row" style={{ alignItems: "center" }}>
        <Radio
          checked={value === "1"}
          id="1"
          value="1"
          onChange={handleChange}
        />
        <Typography type="l1">Radio 1</Typography>
      </div>
      <div className="row" style={{ alignItems: "center" }}>
        <Radio
          checked={value === "2"}
          id="2"
          value="2"
          onChange={handleChange}
        />
        <Typography type="l1">Radio 2</Typography>
      </div>
    </div>
  );
}
```

### Radio customizado

Você também pode criar seu próprio radio customizado.

```tsx live
function example() {
  const [value, setValue] = useState("1");

  function handleChange(e) {
    setValue(e.currentTarget.value);
  }

  return (
    <div className="column">
      <div className="row" style={{ alignItems: "center" }}>
        <Radio
          checked={value === "1"}
          id="1"
          value="1"
          onChange={handleChange}
          activeColor="red"
          inactiveColor="red"
        />
        <Typography type="l1">Radio 1</Typography>
      </div>
      <div className="row" style={{ alignItems: "center" }}>
        <Radio
          checked={value === "2"}
          id="2"
          value="2"
          onChange={handleChange}
          activeColor="green"
        />
        <Typography type="l1">Radio 2</Typography>
      </div>
    </div>
  );
}
```
