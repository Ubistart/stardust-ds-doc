## Uso

Exemplo de uso básico do componente de input:

```tsx live
<Input isFullWidth placeholder="Uso básico" />
```

### Input controlado

```tsx live
function example() {
  const [value, setValue] = React.useState("");
  const handleChange = (event) => setValue(event.target.value);

  return (
    <>
      <Typography mBottom="xxxs">Value: {value}</Typography>
      <Input
        isFullWidth
        value={value}
        onChange={handleChange}
        placeholder="Input controlado"
        iconRight={!!value.length ? <PhosphorIcons.X size={32} /> : null}
        iconRightAction={() => setValue("")}
      />
    </>
  );
}
```

### Input com exemplo de senha

Vamos usar esses componentes para criar um input com uma funcionalidade de mostrar/ocultar senha:

```tsx live
function example(props) {
  const [show, setShow] = React.useState(false);
  const handleClick = () => setShow(!show);

  const canSeePasswordIcon = !show ? (
    <PhosphorIcons.Eye size={32} weight="light" color="white" />
  ) : (
    <PhosphorIcons.EyeSlash size={32} weight="light" />
  );

  return (
    <Input
      isFullWidth
      type={show ? "text" : "password"}
      placeholder="Input com exemplo de senha"
      iconRight={canSeePasswordIcon}
      iconRightAction={handleClick}
    />
  );
}
```

### Input com icones

Você pode adicionar ícones do lado esquerdo e direito ao componente `Input` usando as propriedades iconLeft e iconRight respectivamente.

```tsx live
<div className="column">
  <Input
    isFullWidth
    placeholder="Input com ícone a esquerda"
    iconLeft={
      <PhosphorIcons.UserCircle size={32} weight="fill" color="black" />
    }
  />
  <Input
    isFullWidth
    placeholder="Input com ícone a direita"
    iconRight={
      <PhosphorIcons.UserCircle size={32} weight="fill" color="black" />
    }
  />
</div>
```

### Input com Label e helper text

```tsx live
<Input
  label="Label"
  helperText="Helper Text"
  isFullWidth
  placeholder="Input com label e helper text"
/>
```

### Input com erro

```tsx live
<Input
  isFullWidth
  hasError
  placeholder="Input com erro"
  helperText="Helper Text"
/>
```

### Input customizado

Você também pode criar seu próprio input customizado.

```tsx live
<Input
  isFullWidth
  placeholder="Input customizado"
  bWidth="hairline"
  bColor="black"
  bStyle="dashed"
/>
```
