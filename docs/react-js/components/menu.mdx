Menu é um componente dropdown isolado para facilitar na criação de sidebars.

- **Obs.:** Todos os exemplos são editávies.

<Tabs groupId='menu-component'>
<TabItem value='Uso' label='Uso'>

## Importação

```tsx
import { Menu } from "@stardust-ds/react";
```

## Uso

### Exemplo de utilização.

```tsx live
function example() {
  MENU_DATA.MOCK_MENU_DATA.icon = <PhosphorIcons.User weight="fill" />;
  MENU_DATA.MOCK_MENU_DATA.submenu[0].icon = (
    <PhosphorIcons.Users weight="fill" />
  );
  return <Menu data={MENU_DATA.MOCK_MENU_DATA} />;
}
```

### Customizando ícone de abrir/fechar menu.

```tsx live
<Menu
  data={MENU_DATA.MOCK_MENU_DATA}
  openSubmenuIcon={<PhosphorIcons.Plus weight="fill" color="black" />}
  closeSubmenuIcon={<PhosphorIcons.Minus weight="fill" color="red" />}
/>
```

### Customizando cores.

```tsx live
function example() {
  MENU_DATA.MOCK_MENU_DATA.iconColor = "red";
  return (
    <Box bgColor="#222" bRadius="xs" width="fit-content" overflow="hidden">
      <Menu
        data={MENU_DATA.MOCK_MENU_DATA}
        bgColor="black"
        hoverBgColor="#222"
        typographyProps={{
          color: "white",
        }}
      />
    </Box>
  );
}
```

### Customizando border left.

```tsx live
<Menu
  data={MENU_DATA.MOCK_MENU_DATA}
  borderLeftWidth={4}
  borderLeftColor="black"
/>
```

## Importação

```tsx
import { MenuProps } from "@stardust-ds/react";

interface MenuDataProps {
  title: string;
  // Caso queira realizar o controle de redirecionamento com react-router-dom.
  onClick?: () => MenuDataProps["url"];
  // Utilizará o href caso não seja passado o onClick.
  url?: string;
  icon?: React.ReactNode;
  iconColor?: string;
  submenu?: MenuDataProps[];
}
```
