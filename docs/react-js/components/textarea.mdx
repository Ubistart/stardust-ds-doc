## Importação

```tsx
import { Textarea } from "@stardust-ds/react";
```

## Uso

```tsx live
<Textarea placeholder="Exemplo de placeholder" />
```

### Textarea controlado

```tsx live
function Example() {
  const [value, setValue] = React.useState("");

  function handleInputChange(e) {
    const inputValue = e.target.value;
    setValue(inputValue);
  }

  return (
    <>
      <Typography mBottom="xxxs">Value: {value}</Typography>
      <Textarea
        value={value}
        onChange={handleInputChange}
        placeholder="Exemplo de placeholder"
      />
    </>
  );
}
```

### Textarea desabilitado

```tsx live
<Textarea disabled placeholder="Exemplo de placeholder" />
```

### Textarea com limite de caracteres

```tsx live
<Textarea maxLength={200} placeholder="Exemplo de placeholder" />
```

### Textarea com erro

```tsx live
<Textarea hasError placeholder="Exemplo de placeholder" />
```

### Textarea customizado

Você também pode criar seu próprio textarea customizado.

```tsx live
<div className="row">
  <Textarea
    label="Label"
    labelTypographyProps={{
      color: "red",
    }}
    placeholder="Exemplo de placeholder"
    height={100}
  />
  <Textarea
    helperText="HelperText"
    helperTypographyProps={{
      color: "red",
    }}
    placeholder="Exemplo de placeholder"
    height={100}
  />
  <Textarea
    maxLength={100}
    letterCounterTypography={{
      color: "red",
    }}
    placeholder="Exemplo de placeholder"
    height={100}
  />
  <Textarea
    bColor="red"
    _focus={{ bColor: "blue" }}
    _hover={{ bColor: "green" }}
    placeholder="Exemplo de placeholder"
    height={100}
    maxWidth={200}
  />
</div>
```

Aceita quase todas as propriedades do typographyProps.

Aceita as propriedades de size, width, height, etc...

Obs.: Também é possível passar todas as propriedades citadas nas seguintes props:

```tsx
  _hover={{}}
  _focus={{}}
  _active={{}}
  _disabled={{}}
```
