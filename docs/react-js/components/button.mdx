Componente de botão para executar evento ou ações. Pode ser usado para submeter formulários, abrir links, etc.
**Obs.:** Todos os exemplos são editávies.

<TabItem value='Uso' label='Uso'>

## Uso

```tsx live
<Button>Button</Button>
```

### Variações do botão

Use a propriedade `variant` para alterar o estilo visual do Button. Você pode definir o valor como `primary` e `secondary`.

```tsx live
<div className="row">
  <Button variant="primary">Primary</Button>
  <Button variant="secondary">Button</Button>
</div>
```

### Botão com propriedade `bgColor`

Use a propriedade bgColor para alterar a cor do `Button`. Você pode definir o valor para qualquer uma das cores do seu sistema de design.

```tsx live
function example(props) {
  const { brand } = useTheme();
  return (
    <div className="row">
      <Button bgColor={brand.color.primary.pure}>Primary</Button>
      <Button bgColor={brand.color.secondary.pure}>Secondary</Button>
      <Button bgColor={brand.color.positive.pure}>Positive</Button>
      <Button bgColor={brand.color.negative.pure}>Negative</Button>
    </div>
  );
}
```

### Botão com ícones

Você pode adicionar ícones esquerdo e direito ao componente `Button` usando as propriedades iconLeft e iconRight respectivamente.

```tsx live
<div className="row">
  <Button iconLeft={<DownloadIcon />}>Button</Button>
  <Button iconRight={<DownloadIcon />}>Button</Button>
</div>
```

### Botão customizado

Você também pode criar seu próprio botão customizado.

```tsx live
<div className="row">
  <Button
    typographyProps={{ fontWeight: "light", type: "p2", color: "aqua" }}
    bWidth="hairline"
    bColor="aqua"
    bStyle="solid"
  >
    Button
  </Button>

  <Button
    labelColor="black"
    bgColor="green"
    bWidth="hairline"
    bColor="red"
    bStyle="solid"
  >
    Button
  </Button>
</div>
```
