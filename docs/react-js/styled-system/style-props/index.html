Propriedades de estilização

Propriedades de estilização são maneiras de alterar o estilo de um componente simplesmente passando propriedades para ele. Economizando tempo fornecendo formas abreviadas e úteis de estilizar seus componentes.

Referência​

As tabelas a seguir mostram listas de cada propriedade de estilo e as propriedades dentro de cada grupo.

Margin e Padding​

import { Typography } from "@stardust-ds/react";

// mTop="nano" refere-se ao valor de 8px;
<Typography mTop="nano">Typography</Typography>;

// pTop="nano" refere-se ao valor de 8px;
<Typography pTop="nano">Typography</Typography>;

PropriedadePropriedade CSS
mTopmargin-top
mRightmargin-right
mBottommargin-bottom
mLeftmargin-left
mxmargin-left + margin-right
mymargin-top + margin-bottom
pToppadding-top
pRightpadding-right
pBottompadding-bottom
pLeftpadding-left
pxpadding-left + padding-right
pypadding-top + padding-bottom

Color e background color​

import { Typography, Badge } from "@stardust-ds/react";

<Typography color="red">Typography</Typography>;

<Badge bgColor="red" label="Badge" />;

PropriedadePropriedade CSS
colorcolor
labelColorcolor
bgColorbackground-color
bgOpacityopacity
