Tema padrão

O objeto de tema é onde você define a paleta de cores da sua aplicação, tipografia, espaçamentos, valores de border, width/height, e muito mais.

Cores​

Adicione um objeto theme.colors ou theme.neutrals para fornecer cores para seu projeto. Por padrão, essas cores podem ser referenciadas por color, neutrals, bColor, bgColor, etc...

import { extendTheme } from "@stardust-ds/react";

// theme.ts
export default extendTheme({
  neutrals: {
    gray1: "#CCC",
    pureBlack: "#000",
    pureWhite: "#FFF",
  },

  colors: {
    transparent: "transparent",
    primary: {
      pure: "red",
    },
    // ...
  },
});

Tipografia​

  • fontSize
  • types
    • title (h1-h7)
    • label (l1-l6)
    • text (p1-p4)
