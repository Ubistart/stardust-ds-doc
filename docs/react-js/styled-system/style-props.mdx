Propriedades de estilização são maneiras de alterar o estilo de um componente simplesmente passando propriedades para ele. Economizando tempo fornecendo formas abreviadas e úteis de estilizar seus componentes.

### Referência

As tabelas a seguir mostram listas de cada propriedade de estilo e as propriedades dentro de cada grupo.

### Margin e Padding

```tsx
import { Typography } from "@stardust-ds/react";

// mTop="nano" refere-se ao valor de 8px;
<Typography mTop="nano">Typography</Typography>;

// pTop="nano" refere-se ao valor de 8px;
<Typography pTop="nano">Typography</Typography>;
```

<TableProps
  title=" "
  headers={["Propriedade", "Propriedade CSS"]}
  rows={[
    ["mTop", "margin-top"],
    ["mRight", "margin-right"],
    ["mBottom", "margin-bottom"],
    ["mLeft", "margin-left"],
    ["mx", "margin-left + margin-right"],
    ["my", "margin-top + margin-bottom"],
    ["pTop", "padding-top"],
    ["pRight", "padding-right"],
    ["pBottom", "padding-bottom"],
    ["pLeft", "padding-left"],
    ["px", "padding-left + padding-right"],
    ["py", "padding-top + padding-bottom"],
  ]}
/>
